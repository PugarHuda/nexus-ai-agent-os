# docs/ — Architecture & API Documentation

## Konteks untuk AI Collaborator

Folder ini berisi dokumentasi arsitektur, API reference, dan panduan integrasi untuk Nexus.

## Files

- `architecture.md` — Arsitektur 6-layer Nexus secara detail
- `api-reference.md` — REST API endpoint documentation
- `0g-integration.md` — Detail integrasi setiap komponen 0G
- `demo-script.md` — Script untuk demo video 3 menit (submission requirement)

## Tujuan Dokumentasi
Hackathon submission requirement mewajibkan README yang mencakup:
1. Project overview
2. System architecture diagram / technical description
3. Penjelasan modul 0G yang digunakan
4. Cara modul tersebut mendukung produk
5. Local deployment steps
6. Test account details / faucet instructions

Semua ini harus ada di README.md root + docs/ ini sebagai supplement.
